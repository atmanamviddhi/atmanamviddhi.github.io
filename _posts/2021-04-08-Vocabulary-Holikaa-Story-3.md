### Few Sankrit Sentences 

| Sanskrit Sentence | English Meaning |
|:-------------------:|:---------------:|
| तस्य हृदये | in his heart |
| तस्य कः अर्थः? | What does it mean? |
| तस्य परिहासं शृणु | Listen to this joke! |
| तस्य एकः पुत्रः आसीत् | He had one boy |
| तस्य पुत्री बहु दयावती | His daughter is a very kind person |
| अहं तस्यै बहु कुपितवती | I got really angry with her. |
| | |
| अपूपम् अस्ति | There is cake. |
| अस्ति किल ? | is it? |
| बहु सम्यक् अस्ति | It is very nice. |
| एवमपि अस्ति वा? | Is it like this? |
| अन्तः किम् अस्ति ? | What is inside? |
| तत् सम्यक् अस्ति | That is correct. |
| | |
| मम वाक्यात् | in my words |
| इदम् मम नास्ति | It is not mine. |
| मम गृहसङ्केत | my residential address |
| मम सहकर्मिणः | my colleagues |
| मम हस्तं धर! | Hold my hand! |
| मम वचनं शृणोतु | Please listen to me. |
| सा मम वयस्का एव | She is the same age as me. |
| | |
| केनापि कारणेन | for some reason or other |
| | |
| किमपि चित्रम् ? | What is strange here? |
| तत् किमपि मास्तु | Don't want that. |
| किमपि न भवति | Nothing happens. |
| किमपि उक्तवान् वा? | Did you say anything? |
| गृहे किमपि पठन्ति वा? | Do they read anything at home? |
| अन्यत् किमपि कार्यम् नास्ति | nothing is permanent |
| अहं किमपि न खादितवान् | I didn't eat anything. |
| तस्मिन् अहं किमपि न वदामि | I do not want to say anything in that regard. |
