### Few Sankrit Sentences 

| Sanskrit Sentence | English Meaning |
|:-------------------:|:---------------:|
| स कुत्र गच्छति ? | Where does he go? |
| भवान् / भवती कुत्र गच्छति? | Where are you going? |
| भवन् तत्र कदा गच्छति? | When are you going there? |
| | |
| सः स्वस्य जननीं सर्वदा सत्करोति | He's very good to his mother. |
| कृष्णफलके सा स्वस्याः नाम लिखितवती | She wrote her name on the blackboard. |
| | |
| तस्याः वचनं श्रुत्वा अहं किञ्चित् चिन्ताक्रान्ता जाता | I was a little bit worried by what she said. |
| | |
| भवतः शाला कदा आरब्धा? | When did your school begin? |
| किमर्थं इदानीं अपि कार्यं न आरब्धम्? | Why hasn't the work begun? |
| | |
| वृक्षम् आरुह्य सः दूरे महासरोवरं दृष्टवान् | After climbing up the tree he was able to see the large lake far away. |
| | |
| तस्य गृहे अहं सुन्दरस्य चषकस्य भग्नं कृतवती \| तस्मात् अहं बहु खिन्ना \| |  I broke a beautiful cup in his house. I am not happy about it at all. |
| | | 
| सः सर्वथा अप्रयोजकः | He is good for nothing. |
| | |
| एतावद् अनृतं वदति इति न ज्ञातवती अहम् | I did no know that he would tell a such lie. |
| | |
