### Few Sankrit Sentences 

| Sanskrit Sentence | English Meaning |

| एवम् एव | just |
| सत्यम् एव | indeed |
| किञ्चित् एव | A little |
| पूर्वम् एव | already |
| अद्य एव वा? | Is it today? |
| इदानीम् एव वा? | Is it going to be now? |
| भवान् एव किम् ? | Is that you? |

| न, त्वदीयम् एव | No, it is yours. |
| लवणं एव नास्ति | No salt at all. |
| स्थलम् एव नास्ति | there is no place |
| उक्तम् एव वदति सः | He has been repeating the same thing. / He repeats himself. |
| वयं आगतवन्तः एव | We have already arrived. |
| दृष्टुं एव न शक्यते | Can't see it. |
|   |   |
| कस्यचित् कालस्य अनन्तरम् | after some time | 
| भवती अनन्तरं उपविशतु | You sit down a little later. |

|   |   |
| एतस्य मूल्यं किम् ? | What is the price of this? |
| एतस्य किं कारणम् ? | What is the reason for this? | 
|   |   |
| तदा सः षड्वयस्कः | He was six at the time. |
| पाकः तदानीं एव सिद्धः | Food is ready a while ago itself. |
| दैवेच्छा तदा आसीत्, किं कुर्मः? | It was God's will. What shall we do? |
